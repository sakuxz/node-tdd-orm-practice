# node TDD sample

## Getting Started

Run TDD test
```shell
npm install
npm test
```

Run with development:
```shell
# Back-End development
npm install
npm run dev

# Front-End development
cd static
npm install
npm run dev
```

Production build:
```shell
# Back-End build
npm install
npm run publish

# Front-End build
cd static
npm install
npm run build
```
